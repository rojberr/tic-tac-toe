## tic-tac-toe

Welcome to my tic-tac-toe project. It is a simple but practical program which I can later use to learn even more computer programming concepts and ideas.


## Usage

Enter cells: 
> O_OXXO_XX

---------
| O _ O |
| X X O |
| _ X X |
---------

## Folder Structure

The workspace contains two folders by default, where:

- `src`: the folder to maintain sources
- `lib`: the folder to maintain dependencies
